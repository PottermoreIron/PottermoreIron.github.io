【留学】文书

最近要开始写文书准备cs的申请了。
本来这事儿一开始想着自己就把材料交给中介，让中介来搞这事儿的。
但感觉还是自己盯着，主导权在自己手里比较好，毕竟这样被拒了的话要怪也只能怪自己，也没啥好后悔的了。
简而言之呢，文书分PS和CV。
CV相当于找工作的时候的简历，尽量简单，一页半左右。
PS相当于慷概陈词，写写自己的历程，好好夸夸自己，再夸夸学校。
但我认为最重要的一步是对症下药，不要乱开枪。
什么意思呢？就像高中老师说的，最好的辅导书就是课本教材，标准答案就搁人家学校网站，普通人咱就别玩花的了，认真把高校的研究生专业网站上的要求好好看看，挖掘一波自己的亮点往上面靠，我猜这不一定能保证文书鹤立鸡群，但绝对稳稳当当，稳扎稳打，十拿九稳，稳操胜券。
不然这学校就有毛病，你写都写你要这种人，这种人来了你不要，你搁这儿搁这儿呢。当然，这是我的一家之言，多问问申到dreamy school的学长姐，看看他们的三维怎么样，文书怎么写的，多收集，多参考点新的信息肯定是没问题的。
所以第一步就是分析一波我选的院校和专业了。

先看看哥大有啥项目可以申请
https://www.cs.columbia.edu/education/ms/

有些track凭名字我还真有点看不懂。
Computational Biology先给排除，劝人学计算生物天打雷劈。
Computer Security，应该跟网安有关，我也没这方面的知识储备，不过有空真能业余学学，小时候诸多梦想之一就是当个黑客，不过研究生还是算了。
Foundations of Computer Science，这foundation一出来就有点吓人，果然，我看了看它的课程，什么图论量子计算密码学博弈论，我这菜鸟高攀不上。
Machine Learning，申请主赛道，给个面子，点进去看看Summary和Courses
Summary

这个track呢，就是机器学习，不过会结合许多不同的领域。
Courses

这是必修课程，要求是1A1B或2A，课程还是很多样，理论，信号处理，语言，图像，机器人都有。
提供的选修课程更加多样，也很多，图比较长，我就懒得贴了，不过感觉都蛮有意思。
Natural Language Processing，dreamy track，得好好看看。
Summary

NLP=文字+语言，挺有意思。
Courses

这个track要求直接上三门必修课，不过我看了看它的选修课很有意思，除了正常的自然语言处理和语音处理的，还有UI设计，语言学和心理学，整挺好！不过傻子才学前端。
Network Systems，虽说我计算机网络这门课拿了97分，但我真不太喜欢搞网络的事，这玩意儿要是搞深了学不太懂，pass掉。
Software Systems，这门track搞软件开发的，估计进去就后端了，课程基本都是数据库，云计算，大规模分布式系统，我觉得也行，是可以接受的，毕竟开发个软件真挺有意思的。
Vision, Graphics, Interaction, and Robotics，我认为可以大概率pass，但也不好说，这track涉及到机器人，我直接一个懵逼，硬件，永远的痛，不过它包括了计算机视觉的一些课程。
剩下两个一个是定制学其他领域的东西，一个好像是专门写论文的？跟我都没关系。
我们可以来看看哥大官网上的MS项目的申请要求
https://www.gradengineering.columbia.edu/graduate-admissions/application-requirements
可以看到，哥大要交85美元的申请费

其次，秋季第一轮项目申请的截止时间是明年1月15日之前，第二轮截止时间是明年2月15日之前。

深度其实就是从很多个分支选三门课学习，但我看AI里的NLP全是statistical的，有点懵逼。
而且除广度和深度之外还有选修要求。
